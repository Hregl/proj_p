# 基于受控标志点 PnP 的飞机姿态评估项目计划书

## 一、项目名称

**基于高精度标定板与受控标志点 PnP 的飞机姿态评估实验系统**

---

## 二、项目背景与目标

本项目拟基于拍摄图像评估飞机或飞机模型的空间姿态。前期采用 4K 工业相机、30mm / 50mm / 100mm 三个焦段镜头、高精度标定板和受控标志点，完成 PnP 姿态估计算法的受控实验验证。后期在此基础上引入更高精度的物理姿态测量设备，进一步冲击 **1 角分** 姿态精度目标。

PnP 问题的基本形式是：已知三维点与其二维图像投影的对应关系、相机内参和畸变参数，求解物体相对相机的旋转和平移。OpenCV 文档中也明确说明，`solvePnP` 及相关函数用于根据 `objectPoints`、`imagePoints`、相机内参矩阵和畸变参数估计物体位姿，并最小化重投影误差；其相机坐标系约定为 X 轴向右、Y 轴向下、Z 轴向前。

---

## 三、总体技术路线

项目总体分为两个阶段：

### 3.1 前期阶段：PnP 算法验证阶段

前期不直接宣称系统达到 1 角分绝对精度，而是完成以下目标：

1. 建立基于高精度标定板的受控参考坐标系；
2. 完成 30mm / 50mm / 100mm 三焦段相机标定；
3. 完成标定板单目标 PnP 验证；
4. 完成飞机刚性标志点目标 PnP 验证；
5. 输出飞机相对标定板参考坐标系的姿态；
6. 分析不同焦段、不同相机模型、不同标志点布局对结果的影响；
7. 形成误差链分析，为后期 1 角分精度验证做准备。

### 3.2 后期阶段：1 角分精度验证阶段

后期引入更高等级的独立物理测量设备，例如：

1. 高精度转台；
2. 高精度双轴倾角仪；
3. 自准直仪；
4. 激光跟踪仪；
5. 三坐标或其他计量设备。

后期目标是验证并优化系统，使其姿态角误差向 **1 角分** 目标收敛。

## 四、阶段性精度目标

### 4.1 1 角分定义

1 角分为：

$$
1' = \frac{1}{60}^{\circ} \approx 0.0167^{\circ}
$$

换算为弧度约为：

$$
0.0167^{\circ} \approx 2.91 \times 10^{-4} \text{ rad}
$$

若飞机标志点有效空间跨度为 (L)，则 1 角分对应的等效横向误差约为：

$$
\Delta x \approx L \cdot 2.91 \times 10^{-4}
$$

例如：

|    有效跨度 | 1 角分对应空间误差 |
| ------: | ---------: |
|  300 mm | 约 0.087 mm |
|  500 mm | 约 0.145 mm |
| 1000 mm | 约 0.291 mm |
| 1500 mm | 约 0.436 mm |

因此，若后期希望正式验证 1 角分精度，飞机标志点三维坐标误差、图像检测误差、相机标定误差、相机外参误差、姿态真值误差等综合误差需要控制在亚毫米量级，且最好使参考真值系统精度高于被测系统目标精度。

### 4.2 前期目标

前期阶段的目标不是直接达到 1 角分，而是：

| 指标                 |       前期建议目标 |
| ------------------ | -----------: |
| 标定板 PnP 重投影 RMSE   | ≤ 0.3–0.5 px |
| 飞机标志点 PnP 重投影 RMSE | ≤ 0.5–1.0 px |
| 同一姿态重复性            |    优于 2–5 角分 |
| 坐标系方向判断            |           正确 |
| 三焦段趋势              |          可解释 |
| 前后相机外参漂移           |  可检测、可记录、可剔除 |

若前期重复性已经接近 1–2 角分，说明系统具备后续冲击 1 角分精度的潜力。

## 五、实验设备与条件

### 5.1 已知实验条件

| 项目    | 配置                   |
| ----- | -------------------- |
| 相机    | 4K 工业相机              |
| 镜头    | 30mm、50mm、100mm 三个焦段 |
| 标定板   | 300mm × 250mm 高精度标定板 |
| 标定板精度 | 0.01mm               |
| 标定板图案 | 可确定方向                |
| 标志点方案 | 前期采用受控标志点 PnP        |
| 姿态输出  | 飞机相对地面/标定板参考坐标系的姿态角  |
| 最终目标  | 1 角分姿态精度             |

### 5.2 标定板作用

300mm × 250mm、0.01mm 精度、方向唯一的高精度标定板可用于：

1. 提供前期世界坐标系参考；
2. 提供尺度基准；
3. 确定 X / Y 坐标轴方向；
4. 防止 180° 旋转或镜像歧义；
5. 辅助相机外参求解；
6. 作为前期 PnP 链路验证目标；
7. 作为后续 SfM / 摄影测量点库建立的控制基准。

标定板 0.01mm 几何精度相对于 300mm 有效基线，对应角度量级约为：

$$
\frac{0.01}{300} \text{ rad} \approx 0.0019^{\circ} \approx 0.115'
$$

相对于 250mm 有效基线，对应角度量级约为：

$$
\frac{0.01}{250} \text{ rad} \approx 0.0023^{\circ} \approx 0.137'
$$

因此，标定板自身几何精度足以支撑前期 PnP 算法验证，也具备支撑后期 1 角分目标的尺度基础。但最终系统精度仍取决于相机标定、图像点检测、标定板安装、飞机标志点空间布局和姿态参考系统等完整误差链。

## 六、坐标系定义

### 6.1 相机坐标系 (C)

采用 OpenCV 默认相机坐标系：

| 轴     | 方向   |
| ----- | ---- |
| (X_C) | 图像右方 |
| (Y_C) | 图像下方 |
| (Z_C) | 镜头前方 |

OpenCV 文档中 `solvePnP` 返回的 `rvec` 与 `tvec` 表示从物体坐标系到相机坐标系的变换。

### 6.2 标定板/前期世界坐标系 (G)

前期将标定板坐标系作为参考世界坐标系 (G)：

| 坐标轴   | 定义               |
| ----- | ---------------- |
| (O_G) | 标定板指定原点          |
| (X_G) | 标定板长边方向，300mm 方向 |
| (Y_G) | 标定板短边方向，250mm 方向 |
| (Z_G) | 标定板法向方向          |

由于标定板图案可确定方向，因此 (X_G)、(Y_G) 的方向唯一。

需要注意：前期输出的是**飞机相对标定板参考坐标系的姿态**。若标定板与真实地面坐标系完全一致，则可近似视为飞机相对地面的姿态；若标定板安装存在倾斜，则需要额外测量标定板坐标系到真实地面坐标系的变换。

### 6.3 飞机本体坐标系 (B)

建议采用如下定义：

| 轴     | 方向   |
| ----- | ---- |
| (X_B) | 机头方向 |
| (Y_B) | 右翼方向 |
| (Z_B) | 机腹方向 |

飞机姿态最终表示为：

$$
{}^{G} R_{B}
$$

即飞机本体坐标系 (B) 相对于参考世界坐标系 (G) 的旋转。

## 七、姿态转换关系

PnP 直接求解的是飞机相对相机的位姿：

$$
{}^{C}T_{B}
$$

即：

$$
{}^{C}T_{B}
$$

标定板 PnP 可求得标定板/世界坐标系相对相机的位姿：

$$
{}^{C}T_{G}
$$

取逆可得相机相对世界坐标系的位姿：

$$
{}^{G}T_{C} = \left({}^{C}T_{G}\right)^{-1}
$$

因此飞机相对世界坐标系的姿态为：

$$
{}^{G}R_{B} = {}^{G}R_{C} \cdot {}^{C}R_{B}
$$

该公式是整个系统坐标链的核心。

前期如果只关心姿态角，可以弱化平移量分析，但平移量仍参与 PnP 求解和重投影误差评价。

## 八、相机标定方案

### 8.1 三焦段独立标定

30mm、50mm、100mm 三个焦段必须分别标定，不能共用同一组内参。

分别输出：

```text
camera_30mm.yaml
camera_50mm.yaml
camera_100mm.yaml
```

每个文件至少包括：

```text
image_width
image_height
camera_matrix K
distortion_coefficients
calibration_reprojection_error
calibration_date
lens_id
focus_distance
aperture
```

OpenCV 相机标定教程说明，标定需要已知几何图案的三维点和图像二维点，并估计相机矩阵、畸变参数、旋转和平移向量；相机矩阵包含 (f_x)、(f_y)、(c_x)、(c_y)，畸变参数通常包括径向和切向畸变项。

### 8.2 标定采集要求

每个焦段建议采集：

| 项目    | 要求                  |
| ----- | ------------------- |
| 标定图数量 | 30–50 张             |
| 姿态    | 正视、斜视、俯仰、左右倾斜       |
| 图像覆盖  | 中心、四角、边缘均覆盖         |
| 距离    | 覆盖正式实验距离附近          |
| 焦距    | 固定                  |
| 对焦    | 固定                  |
| 光圈    | 固定                  |
| 图像分辨率 | 与正式实验一致             |
| 图像格式  | PNG / BMP / TIFF 优先 |

OpenCV 教程中也指出，为了获得更好标定结果，需要多张已知图案图像，并使用已知真实坐标与图像坐标对应关系求解相机参数。

## 九、相机模型比较方案

### 9.1 比较目的

不同相机模型会影响三维点重建、畸变校正和 PnP 姿态结果。前期应将不同相机模型作为系统误差分析内容，而不是把不同模型结果直接当作多套“真值”。

### 9.2 建议比较模型

每个焦段建议比较：

| 模型            | 用途           |
| ------------- | ------------ |
| PINHOLE       | 低畸变或已去畸变图像基线 |
| SIMPLE_RADIAL | 简化径向畸变模型     |
| OPENCV        | 主推荐模型        |
| FULL_OPENCV   | 复杂畸变对照模型     |

COLMAP 文档建议根据图像畸变复杂度选择相机模型；`PINHOLE` 适合已去畸变图像，`SIMPLE_RADIAL` / `RADIAL` 是简化径向模型，`OPENCV` / `FULL_OPENCV` 更适合已有先验标定参数的情形。文档同时提醒，模型过于复杂或每张图独立估计内参可能导致参数估计失败。

### 9.3 选择原则

最终模型选择不只看重投影误差，还应综合：

1. 标定板尺度残差；
2. 检查点误差；
3. 畸变参数稳定性；
4. 三焦段姿态重复性；
5. 飞机刚体点间距离稳定性；
6. 坐标轴回投影是否合理。

## 十、受控标志点设计

### 10.1 标定板标志点

标定板图案点坐标由厂家或设计文件给出，作为前期世界坐标系控制点。

要求：

1. 点编号明确；
2. 点方向唯一；
3. 点间距已知；
4. 图案点坐标以 mm 为单位；
5. 点坐标文件不可随实验过程更改。

### 10.2 飞机标志点

飞机或飞机刚性模型上布置受控标志点，用于 PnP 求解飞机位姿。

建议：

| 区域      |  点数 |
| ------- | --: |
| 机头      | 2–4 |
| 左翼      | 4–8 |
| 右翼      | 4–8 |
| 机尾/平尾   | 4–8 |
| 垂尾/机背   | 3–6 |
| 机腹/下方结构 | 3–6 |

关键原则：

1. 标志点不能全部共面；
2. 点要覆盖飞机前后、左右、上下；
3. 左右翼要能区分，避免镜像混淆；
4. 每张图至少可见 8 个以上有效点；
5. 有效空间跨度越大，姿态角越稳定；
6. 标志点应刚性固定，不得随拍摄过程移动。

前期可以使用未编码圆形靶标配合少量方向编码点或人工编号方式。未编码圆点有利于高精度中心定位，但必须解决跨视角点号对应问题。

## 十一、前期实验流程

### 11.1 阶段 1：标定板单目标 PnP 验证

目的：先验证相机标定、畸变模型、PnP 求解、重投影误差计算和坐标轴可视化是否正常。

流程：

1. 固定相机；
2. 放置高精度标定板；
3. 检测标定板图案点；
4. 根据已知 3D 点与图像 2D 点求解 PnP；
5. 输出标定板相对相机的姿态；
6. 对标定板坐标轴进行图像回投影；
7. 计算重投影 RMSE；
8. 对 30mm / 50mm / 100mm 分别重复。

验收指标：

| 指标        |           要求 |
| --------- | -----------: |
| 检测点数量     |  满足 PnP 输入要求 |
| 重投影 RMSE  | ≤ 0.3–0.5 px |
| 坐标轴方向     |      与图案方向一致 |
| 重复拍摄姿态标准差 | 越小越好，目标优于数角分 |

### 11.2 阶段 2：飞机刚性标志点 PnP 验证

目的：验证飞机目标位姿估计流程。

流程：

1. 建立飞机本体坐标系 (B)；
2. 建立飞机标志点三维坐标表；
3. 拍摄飞机标志点目标；
4. 检测图像 2D 标志点；
5. 求解飞机相对相机姿态 ({}^C R_B)；
6. 同时通过标定板求解相机相对参考世界坐标系 ({}^G R_C)；
7. 合成得到飞机相对参考世界坐标系姿态：

$$
{}^G R_B = {}^G R_C \cdot {}^C R_B
$$

8. 输出 yaw / pitch / roll；
9. 计算重投影误差；
10. 统计重复性。

### 11.3 阶段 3：三焦段对比实验

对 30mm、50mm、100mm 进行同样流程。

比较内容：

| 指标          | 30mm | 50mm | 100mm |
| ----------- | ---: | ---: | ----: |
| 标定板重投影 RMSE |      |      |       |
| 飞机点重投影 RMSE |      |      |       |
| yaw 重复性     |      |      |       |
| pitch 重复性   |      |      |       |
| roll 重复性    |      |      |       |
| 可见标志点数      |      |      |       |
| 检测失败率       |      |      |       |
| 对焦敏感性       |      |      |       |
| 视场完整性       |      |      |       |

预期上，30mm 视场大、容错高，但目标像素跨度小、畸变影响更明显；100mm 目标像素跨度大，理论角度分辨率更高，但对对焦、震动、视场范围要求更严格；50mm 可能具有较好的综合稳定性。

## 十二、姿态角定义与输出

建议采用 ZYX 欧拉角定义：

$$
{}^G R_B = R_z(yaw) \cdot R_y(pitch) \cdot R_x(roll)
$$

姿态角含义：

| 角度    | 含义                   |
| ----- | -------------------- |
| yaw   | 机头在参考平面内相对 (X_G) 的偏航 |
| pitch | 机头上仰/下俯              |
| roll  | 左右翼绕机身轴的横滚           |

前期必须做符号验证：

1. 只改变 yaw，检查输出 yaw 是否变化；
2. 只改变 pitch，检查输出 pitch 是否变化；
3. 只改变 roll，检查输出 roll 是否变化；
4. 坐标轴回投影方向必须与实物一致。

## 十四、SfM / 摄影测量点库的定位

### 14.1 前期定位

多视角 SfM / 近景摄影测量可用于建立参考三维点库，但不作为最终绝对真值来源。

可建立两类点库：

```text
ground_points_G.csv
aircraft_points_B.csv
```

其中：

1. `ground_points_G.csv`：标定板或地面控制点在参考世界坐标系 (G) 下的三维坐标；
2. `aircraft_points_B.csv`：飞机标志点在飞机本体坐标系 (B) 下的三维坐标。

COLMAP 文档说明，SfM 是从多视角重叠图像中恢复三维结构和相机位姿的过程，输入是同一对象不同视角的图像，输出包括对象三维重建以及相机内外参。

### 14.2 使用边界

SfM / 摄影测量在本项目中的作用是：

1. 辅助建立飞机标志点三维坐标；
2. 检查标志点是否非共面；
3. 检查标志点空间跨度；
4. 评估不同相机模型对重建结果的影响；
5. 为后期精度分析提供参考。

但前期不应将其称为“绝对真值数据库”，更合适的说法是：

```text
基于标定板约束的参考三维点库
```

### 14.3 采集要求

SfM / 摄影测量图像采集应满足：

1. 多视角；
2. 高重叠；
3. 避免纯旋转拍摄；
4. 避免反光和运动模糊；
5. 每个目标点至少被 3 张以上图像看到，更多更好；
6. 控制照明一致性；
7. 每个焦段分别建模或明确相机内参共享关系。

COLMAP 文档对受控拍摄也提出建议：应保证纹理、相似光照、高视觉重叠，并从不同视角拍摄；同一物理相机、同一焦距和相同分辨率/畸变特性可以共享内参。

## 十五、评价指标

### 15.1 图像层指标

| 指标      | 含义           |
| ------- | ------------ |
| 点检测成功率  | 标志点是否稳定被检测   |
| 图像点定位残差 | 点检测精度间接反映    |
| 可见点数量   | 参与 PnP 的有效点数 |
| 遮挡率     | 影响 PnP 稳定性   |

### 15.2 PnP 层指标

| 指标         | 含义           |
| ---------- | ------------ |
| 重投影 RMSE   | PnP 几何一致性    |
| 最大重投影误差    | 检查异常点        |
| RANSAC 内点数 | 稳健性          |
| 内点比例       | 误匹配程度        |
| 姿态重复性      | 同一姿态重复拍摄的标准差 |

OpenCV 文档中也说明，迭代式 PnP 会最小化图像观测点与通过 `projectPoints` 投影的三维点之间的重投影误差；`solvePnPRansac` 则用于在存在外点时进行稳健估计，姿态精修可用 `solvePnPRefineLM` 或 `solvePnPRefineVVS`。

### 15.3 坐标链指标

| 指标        | 含义                     |
| --------- | ---------------------- |
| 标定板坐标轴回投影 | 检查世界坐标系方向              |
| 飞机坐标轴回投影  | 检查飞机姿态方向               |
| 前后标定板外参漂移 | 检查相机是否移动               |
| 三焦段姿态差异   | 检查镜头/模型影响              |
| 单轴变化趋势    | 检查 yaw/pitch/roll 解耦情况 |

### 15.4 后期真值指标

后期引入独立物理真值设备后，应增加：

| 指标       | 含义                         |
| -------- | -------------------------- |
| yaw 误差   | (yaw_{est} - yaw_{gt})     |
| pitch 误差 | (pitch_{est} - pitch_{gt}) |
| roll 误差  | (roll_{est} - roll_{gt})   |
| 姿态总角误差   | 旋转矩阵整体误差                   |
| 误差均值     | 系统偏差                       |
| 误差标准差    | 重复性                        |
| 最大绝对误差   | 最坏情况                       |
| 合格率      | 满足 1 角分阈值比例                |

## 十八、主要风险与控制措施

### 18.1 标定板安装误差

风险：

标定板自身精度很高，但安装姿态误差可能直接影响参考坐标系。

措施：

1. 记录标定板安装方式；
2. 固定标定板，不得移动；
3. 实验前后重复拍摄标定板；
4. 后期引入水平仪、倾角仪或转台测量标定板与真实地面坐标系关系。

### 18.2 单块平面板 Z 向约束不足

风险：

单块平面标定板主要提供平面内尺度和方向，对三维空间的 Z 向约束较弱。

措施：

1. 前期接受其作为算法验证基准；
2. 后期增加垂直板、L 型标定架或非共面控制点；
3. 使用检查点验证三维精度。

### 18.3 飞机标志点共面或跨度不足

风险：

若飞机标志点接近共面或空间跨度太小，姿态估计尤其是 pitch / roll 稳定性会下降。

措施：

1. 增加机背、机腹、垂尾、下方结构点；
2. 扩大前后、左右、上下跨度；
3. 避免左右完全对称无编号布局；
4. 统计不同标志点组合下的姿态稳定性。

### 18.4 相机模型过拟合

风险：

复杂相机模型可能降低重投影误差，但引入几何变形或参数不稳定。

措施：

1. 不只看重投影误差；
2. 比较尺度残差、重复性和姿态稳定性；
3. 优先选择物理合理且稳定的模型；
4. 每个焦段独立评估。

### 18.5 100mm 镜头实验敏感

风险：

100mm 焦段理论像素跨度大，但对震动、对焦、景深和视场范围更敏感。

措施：

1. 使用高稳定支架；
2. 固定对焦和光圈；
3. 保证目标完整入镜；
4. 每组多拍，剔除异常样本。

## 二十、最终项目口径

本项目前期采用 300mm × 250mm、几何精度 0.01mm、图案方向唯一的高精度标定板作为受控实验参考基准，结合 4K 工业相机和 30mm / 50mm / 100mm 三焦段镜头，完成基于受控标志点的 PnP 姿态估计算法验证。前期重点验证 PnP 解算链路、相机标定、坐标系转换、三焦段差异、重投影误差和姿态重复性。

本阶段不直接宣称系统达到 1 角分绝对姿态精度，而是将 1 角分作为最终优化目标。后续阶段将在现有系统基础上引入高精度转台、倾角仪、自准直仪或其他独立计量设备，对姿态估计结果进行正式真值验证，并进一步优化系统误差链，最终冲击 1 角分姿态精度目标。

## 二十一、参考依据

1. OpenCV `solvePnP` 文档：PnP 通过 3D–2D 对应关系、相机内参和畸变参数估计物体相对相机位姿，并最小化重投影误差。
2. OpenCV 相机标定文档：相机标定用于估计相机矩阵、畸变参数和外参，标定依赖已知几何图案的三维点与图像二维点对应关系。
3. COLMAP SfM 文档：SfM 从多视角重叠图像中恢复三维结构与相机内外参。
4. COLMAP 相机模型文档：相机模型复杂度需要与畸变情况匹配，过于复杂或每张图独立估计内参可能导致参数不稳定。
