# 基于高精度标定板与受控标志点 PnP 的飞机姿态评估项目计划书

## 1. 项目概述

### 1.1 项目名称

**基于高精度标定板与受控标志点 PnP 的飞机姿态评估实验系统**

### 1.2 项目背景

本项目拟通过拍摄图像评估飞机或飞机模型的空间姿态。前期采用 4K 工业相机、30mm / 50mm / 100mm 三个焦段镜头、300mm × 250mm 高精度方向标定板和受控标志点，完成 PnP 姿态估计算法的实验验证。

PnP 的基本任务是根据一组三维点及其二维图像投影、相机内参和畸变参数，估计物体相对相机的旋转和平移。OpenCV 官方文档说明，`solvePnP` 及相关函数用于根据 3D–2D 点对应关系、相机内参矩阵和畸变参数估计物体姿态，并最小化重投影误差；其相机坐标系约定为 X 轴向右、Y 轴向下、Z 轴向前。

### 1.3 项目阶段定位

本项目分为两个阶段：

| 阶段   | 目标            | 真值/参考来源            | 结论口径            |
| ---- | ------------- | ------------------ | --------------- |
| 前期阶段 | 验证 PnP 姿态估计链路 | 高精度标定板建立参考坐标系      | 输出参考坐标系下的姿态估计结果 |
| 后期阶段 | 冲击 1 角分姿态精度   | 转台、倾角仪、自准直仪等独立物理设备 | 验证绝对姿态精度        |

前期阶段的核心目标不是直接证明系统达到 1 角分绝对精度，而是完成：

1. 相机标定流程验证；
2. 标定板 PnP 验证；
3. 飞机受控标志点 PnP 验证；
4. 坐标链转换验证；
5. 三焦段成像条件对比；
6. 姿态重复性和重投影误差分析；
7. 为后期引入物理真值设备冲击 1 角分精度建立实验基础。

---

## 2. 项目目标

### 2.1 总体目标

建立一套基于高精度标定板和受控标志点的图像姿态评估实验系统，完成飞机模型在参考世界坐标系下的姿态估计，并评估不同焦段、不同相机模型和不同标志点布局对姿态结果的影响。

### 2.2 前期目标

前期目标如下：

1. 完成 30mm / 50mm / 100mm 三个焦段的相机标定；
2. 建立标定板参考世界坐标系；
3. 使用标定板完成 PnP 位姿求解验证；
4. 使用飞机模型或刚性飞机标志点架完成 PnP 位姿求解验证；
5. 将飞机相对相机的姿态转换到标定板参考坐标系；
6. 输出 yaw / pitch / roll；
7. 统计重投影误差、姿态重复性和三焦段差异；
8. 形成误差分析报告和后期升级建议。

### 2.3 后期目标

后期目标是在前期系统稳定的基础上，引入高精度物理计量设备，正式验证姿态估计精度，并将系统优化目标设为：

$$
1' = \frac{1}{60}^{\circ} \approx 0.0167^{\circ}
$$

即姿态角误差最终目标为 **1 角分**。

### 3.2 标定板能力分析

标定板几何精度为 0.01mm。若以 300mm 有效基线估算，其几何误差对应的角度量级为：

$$
\theta \approx \frac{0.01}{300} \ \text{rad}
$$

换算为角度：

$$
\frac{0.01}{300} \ \text{rad} \approx 0.0019^{\circ} \approx 0.115'
$$

若以 250mm 有效基线估算：

$$
\frac{0.01}{250} \ \text{rad} \approx 0.0023^{\circ} \approx 0.137'
$$

因此，标定板自身几何精度明显优于 1 角分量级，足以作为前期 PnP 算法验证和参考坐标系建立的核心基准。

但需要注意：

> 标定板自身精度足够，不代表整个姿态估计系统已经达到 1 角分精度。系统最终精度还取决于相机标定、镜头畸变、图像点检测、标志点安装、飞机点库、标定板安装姿态、相机稳定性和后期真值设备精度。

## 4. 技术路线

### 4.1 总体路线

前期技术路线如下：

```text
三焦段相机标定
    ↓
标定板点检测
    ↓
标定板 PnP 验证
    ↓
飞机受控标志点布置
    ↓
飞机标志点 2D 检测 / 标注
    ↓
飞机 PnP 姿态估计
    ↓
标定板坐标链转换
    ↓
输出飞机相对参考世界坐标系的 yaw / pitch / roll
    ↓
重投影误差、重复性、三焦段差异分析
```

### 4.2 前期实验逻辑

前期不直接依赖物理转台作为真值，而是先用高精度标定板建立可控参考世界坐标系。该阶段主要验证：

1. PnP 算法链路是否可行；
2. 坐标系定义是否正确；
3. 相机标定是否稳定；
4. 飞机受控标志点是否能稳定解算姿态；
5. 三个焦段中哪个更适合后续高精度实验；
6. 当前系统主要误差来自哪里。

### 4.3 后期实验逻辑

后期引入独立物理设备，包括但不限于：

1. 高精度 yaw 转台；
2. 高精度双轴倾角仪；
3. 三轴姿态转台；
4. 自准直仪；
5. 激光跟踪仪；
6. 三坐标测量设备。

后期用于验证：

$$
yaw_{err} = yaw_{est} - yaw_{gt}
$$

$$
pitch_{err} = pitch_{est} - pitch_{gt}
$$

$$
roll_{err} = roll_{est} - roll_{gt}
$$

并以 1 角分作为最终优化目标。

## 5. 坐标系定义

坐标系定义是本项目的核心。前期必须先冻结坐标系定义，再开展实验。

### 5.1 相机坐标系 C

采用 OpenCV 默认相机坐标系：

| 坐标轴   | 方向   |
| ----- | ---- |
| (X_C) | 图像右方 |
| (Y_C) | 图像下方 |
| (Z_C) | 镜头前方 |

OpenCV 文档说明，`solvePnP` 返回的 `rvec` 和 `tvec` 表示将物体坐标系中的三维点变换到相机坐标系中的旋转和平移。

### 5.2 标定板参考世界坐标系 G

前期将标定板坐标系作为参考世界坐标系 (G)。

建议定义如下：

| 坐标轴   | 定义             |
| ----- | -------------- |
| (O_G) | 标定板指定原点        |
| (X_G) | 标定板 300mm 长边方向 |
| (Y_G) | 标定板 250mm 短边方向 |
| (Z_G) | 标定板法向方向        |

由于标定板图案可确定方向，因此可避免 180° 旋转歧义、镜像歧义和坐标轴方向不确定问题。

### 5.3 飞机本体坐标系 B

建议定义如下：

| 坐标轴   | 方向   |
| ----- | ---- |
| (X_B) | 机头方向 |
| (Y_B) | 右翼方向 |
| (Z_B) | 机腹方向 |

飞机姿态最终表示为飞机本体坐标系 (B) 相对于参考世界坐标系 (G) 的旋转：

$$
{}^{G}R_{B}
$$

## 6. 坐标链与姿态解算关系

### 6.1 标定板 PnP

标定板上的三维点坐标已知，记为：

$$
P_G = \left[X_G, Y_G, Z_G\right]^T
$$

通过图像检测得到二维点：

$$
p = \left[u, v\right]^T
$$

标定板 PnP 可求得标定板参考世界坐标系相对相机坐标系的位姿：

$$
{}^{C}T_{G}
$$

其中旋转部分为：

$$
{}^{C}R_{G}
$$

平移部分为：

$$
{}^{C}t_{G}
$$

### 6.2 飞机 PnP

飞机标志点在飞机本体坐标系下的三维坐标已知，记为：

$$
P_B = \left[X_B, Y_B, Z_B\right]^T
$$

通过图像检测得到对应二维点：

$$
p = \left[u, v\right]^T
$$

飞机 PnP 可求得飞机本体坐标系相对相机坐标系的位姿：

$$
{}^{C}T_{B}
$$

其中旋转部分为：

$$
{}^{C}R_{B}
$$

### 6.3 姿态合成

标定板 PnP 得到的是：

$$
{}^{C}T_{G}
$$

要求飞机相对参考世界坐标系的姿态，需要先求逆：

$$
{}^{G}T_{C} = \left({}^{C}T_{G}\right)^{-1}
$$

因此：

$$
{}^{G}R_{B} = {}^{G}R_{C} \cdot {}^{C}R_{B}
$$

该公式是前期姿态输出的核心。

### 6.4 欧拉角输出

建议采用 ZYX 欧拉角定义：

$$
{}^{G}R_{B} = R_z(yaw) \cdot R_y(pitch) \cdot R_x(roll)
$$

输出：

```text
yaw_deg
pitch_deg
roll_deg
```

需要通过单轴实验验证符号：

1. 只改变 yaw，主要变化应出现在 yaw；
2. 只改变 pitch，主要变化应出现在 pitch；
3. 只改变 roll，主要变化应出现在 roll；
4. 坐标轴回投影方向必须与实物一致。

## 8. 飞机标志点布置方案

### 8.1 布点原则

飞机目标点的布局直接决定 PnP 姿态稳定性。前期建议采用刚性飞机标志点架，而不是只依赖飞机自然外形点。

布点原则：

1. 点尽量分散；
2. 点不能全部共面；
3. 点应覆盖前后方向；
4. 点应覆盖左右方向；
5. 点应覆盖上下方向；
6. 点位应具有唯一对应关系；
7. 每张图至少应可见 8 个有效点；
8. 有效空间跨度越大，姿态估计越稳定。

### 8.2 推荐点位数量

| 区域        | 推荐点数 | 作用        |
| --------- | ---: | --------- |
| 机头        |  2–4 | 约束机头方向    |
| 左翼        |  4–8 | 约束横滚和左右方向 |
| 右翼        |  4–8 | 约束横滚和左右方向 |
| 机尾 / 平尾   |  4–8 | 约束偏航和俯仰   |
| 垂尾 / 机背   |  3–6 | 提供高度方向约束  |
| 机腹 / 下方结构 |  3–6 | 增强非共面性    |

### 8.3 标志点类型

前期可采用：

```text
未编码圆形靶标 + 少量方向编码点 + 人工/半自动编号
```

优点：

1. 未编码圆点中心定位精度较高；
2. 圆点制作简单；
3. 大量布点成本低；
4. 少量方向编码点可辅助建立点号和方向。

风险：

1. 未编码点本身没有 ID；
2. 多视角或遮挡情况下容易点号混淆；
3. PnP 必须依赖正确的 3D–2D 点对应关系；
4. 点号错误会导致姿态解算失败。

因此，前期必须实现人工复核或半自动编号功能。

## 9.2 阶段 1：三焦段相机标定

### 9.2.1 目标

获得 30mm、50mm、100mm 三个焦段对应的相机内参和畸变参数。

OpenCV 相机标定文档说明，相机标定用于获得相机内参、外参和畸变参数；内参包括焦距 (f_x, f_y) 和主点 (c_x, c_y)，畸变参数通常包括径向畸变和切向畸变。

### 9.2.2 操作内容

每个焦段分别采集：

| 项目     | 要求                  |
| ------ | ------------------- |
| 标定图数量  | 30–50 张             |
| 图像区域覆盖 | 中心、四角、边缘均覆盖         |
| 姿态     | 正视、斜视、俯仰、旋转         |
| 距离     | 覆盖正式实验距离附近          |
| 对焦     | 与正式实验一致             |
| 光圈     | 与正式实验一致             |
| 分辨率    | 与正式实验一致             |
| 图像格式   | PNG / BMP / TIFF 优先 |

OpenCV 文档指出，为了求取相机参数，需要提供已知几何图案的多张样本图像；官方教程中也提到，较好的结果至少需要多张测试图案，且标定结果可以通过重投影误差评估。

### 9.2.3 输出

```text

camera_100mm.yaml
```

## 9.3 阶段 2：标定板单目标 PnP 验证

## 9.4 阶段 3：飞机目标 PnP 验证

### 9.4.1 目标

验证飞机受控标志点目标能否稳定完成姿态估计。

### 9.4.2 操作内容

1. 安装飞机标志点；
2. 建立飞机本体坐标系 (B)；
3. 建立 `aircraft_points_B.csv`；
4. 拍摄飞机目标图像；
5. 检测或人工标注飞机标志点二维坐标；
6. 建立飞机 3D 点与图像 2D 点对应；
7. 执行飞机 PnP；
8. 输出 ({}^{C}R_B)、({}^{C}t_B)；
9. 计算飞机点重投影误差；
10. 回投影飞机坐标轴；
11. 检查机头、右翼、机腹方向是否正确。

### 9.4.3 输出

```text
aircraft_pose_camera_results.csv
aircraft_reprojection_errors.csv
aircraft_axis_visualization/
aircraft_point_detection_visualization/
```

### 9.4.4 验收标准

| 指标          |         建议标准 |
| ----------- | -----------: |
| 有效点数量       |        ≥ 8 个 |
| 飞机 PnP 成功率  |        ≥ 95% |
| 飞机点重投影 RMSE | ≤ 0.5–1.0 px |
| 坐标轴回投影      |         方向正确 |
| 左右翼点号       |          无混淆 |
| 异常点         |      可识别、可剔除 |

## 9.5 阶段 4：标定板 + 飞机联合姿态输出

### 9.5.1 目标

输出飞机相对标定板参考世界坐标系的姿态。

### 9.5.2 操作方式

存在两种实验方式：

#### 方式 A：标定板与飞机同时入镜

优点：

1. 每张图都能同时估计相机外参和飞机位姿；
2. 可抵消相机轻微移动；
3. 坐标链更稳。

流程：

```text
检测标定板点
    ↓
求解标定板相对相机位姿
    ↓
检测飞机标志点
    ↓
求解飞机相对相机位姿
    ↓
合成飞机相对参考世界坐标系姿态
```

#### 方式 B：标定板与飞机分开拍摄

适用于视场不足或 100mm 镜头无法同时拍摄的情况。

流程：

```text
实验前拍标定板
    ↓
求相机相对参考世界坐标系外参
    ↓
拍摄飞机目标
    ↓
求飞机相对相机位姿
    ↓
实验后再拍标定板
    ↓
检查相机外参是否漂移
```

如果前后标定板外参变化超过设定阈值，则该组实验应标记为异常。

### 9.5.3 输出

```text
final_pose_results.csv
final_pose_repeatability.csv
lens_comparison_summary.csv
```

字段示例：

```csv
image_id,lens,camera_model,yaw_deg,pitch_deg,roll_deg,board_rmse_px,aircraft_rmse_px,quality_flag
```

## 9.6 阶段 5：三焦段对比实验

### 9.6.1 目标

比较 30mm、50mm、100mm 三个焦段对姿态估计稳定性的影响。

### 9.6.2 姿态序列

前期建议先做单轴变化：

```text
yaw:   -10°, -5°, 0°, 5°, 10°
pitch: -5°, 0°, 5°
roll:  -5°, 0°, 5°
```

再做少量组合姿态：

```text
yaw = 10°,  pitch = 5°,  roll = -5°
yaw = -10°, pitch = 5°,  roll = 5°
yaw = 5°,   pitch = -5°, roll = 10°
```

前期这些角度可由简易角度夹具或普通旋转结构提供，只用于观察趋势和符号，不作为最终高精度真值。

### 9.6.3 每组拍摄数量

每个焦段、每个姿态建议拍摄：

```text
5–10 张图像
```

每组实验前后拍摄标定板，用于检查相机外参是否漂移。

### 9.6.4 对比指标

| 指标        | 30mm | 50mm | 100mm |
| --------- | ---: | ---: | ----: |
| 标定板 RMSE  |      |      |       |
| 飞机点 RMSE  |      |      |       |
| yaw 重复性   |      |      |       |
| pitch 重复性 |      |      |       |
| roll 重复性  |      |      |       |
| 点检测成功率    |      |      |       |
| PnP 失败率   |      |      |       |
| 对焦敏感性     |      |      |       |
| 视场完整性     |      |      |       |
| 综合评价      |      |      |       |

### 9.6.5 预期现象

| 焦段    | 预期特点                               |
| ----- | ---------------------------------- |
| 30mm  | 视场大，容易拍全目标，但目标像素跨度较小，畸变影响更明显       |
| 50mm  | 视场和像素跨度折中，可能具有较好综合稳定性              |
| 100mm | 目标像素跨度大，理论角度分辨率高，但对对焦、震动、景深和视场要求更高 |

## 13. 评价指标体系

### 13.1 图像检测指标

| 指标     | 含义                       |
| ------ | ------------------------ |
| 点检测成功率 | 检测到的有效点数量占应检测点数量比例       |
| 点号正确率  | 2D 点 ID 与 3D 点 ID 是否正确对应 |
| 可见点数量  | 当前图像可用于 PnP 的点数量         |
| 点分布范围  | 点在图像中是否覆盖足够区域            |

### 13.2 PnP 几何指标

| 指标         | 含义         |
| ---------- | ---------- |
| 重投影 RMSE   | PnP 几何一致性  |
| 最大单点误差     | 判断异常点      |
| RANSAC 内点数 | 稳健性        |
| 内点比例       | 外点比例       |
| 坐标轴回投影     | 判断姿态方向是否正确 |

### 13.3 姿态稳定性指标

| 指标        | 含义                |
| --------- | ----------------- |
| yaw 标准差   | 同一姿态重复拍摄 yaw 波动   |
| pitch 标准差 | 同一姿态重复拍摄 pitch 波动 |
| roll 标准差  | 同一姿态重复拍摄 roll 波动  |
| 三焦段差异     | 不同焦段下姿态结果差异       |
| 单轴变化趋势    | 姿态角是否随实验设置合理变化    |

### 13.4 前期建议阈值

| 指标       |     合格标准 |     优秀标准 |
| -------- | -------: | -------: |

| 坐标轴方向    |       正确 |    正确且稳定 |

## 15. 风险分析与控制措施

### 15.1 标定板安装误差

| 内容 | 说明                                  |
| -- | ----------------------------------- |
| 风险 | 标定板自身精度高，但安装姿态误差会影响参考坐标系            |
| 影响 | 输出姿态相对标定板正确，但不一定相对真实地面正确            |
| 控制 | 记录安装方式；前后拍摄标定板；后期用水平仪或倾角仪测量标定板与地面关系 |

### 15.2 飞机标志点共面

| 内容 | 说明                     |
| -- | ---------------------- |
| 风险 | 标志点集中在一个平面             |
| 影响 | pitch / roll / 深度方向不稳定 |
| 控制 | 增加机背、机腹、垂尾、下方支架点       |

### 15.3 点号错误

| 内容 | 说明                       |
| -- | ------------------------ |
| 风险 | 未编码圆点容易编号错误              |
| 影响 | PnP 姿态严重错误               |
| 控制 | 增加方向编码点；人工复核；异常点剔除；可视化点号 |

### 15.4 相机模型不合适

| 内容 | 说明                                |
| -- | --------------------------------- |
| 风险 | 模型过简或过复杂都会影响姿态结果                  |
| 影响 | 重投影误差小但姿态不稳定，或畸变补偿不足              |
| 控制 | 比较多种模型；不只看 RMSE；看重复性、尺度一致性和坐标轴回投影 |

### 15.5 100mm 镜头敏感

| 内容 | 说明                        |
| -- | ------------------------- |
| 风险 | 100mm 对焦、震动、景深、视场更敏感      |
| 影响 | 理论精度高，但实验失败率可能更高          |
| 控制 | 使用刚性支架；固定对焦；增加光照；保证目标完整入镜 |

### 15.6 前期误称“真值”

| 内容 | 说明                                    |
| -- | ------------------------------------- |
| 风险 | 前期标定板参考坐标系被误写为绝对真值                    |
| 影响 | 项目结论不严谨                               |
| 控制 | 前期统一称为“参考坐标系”或“阶段性参考基准”；后期再引入独立物理真值设备 |

## 17. 后期 1 角分精度验证路线

### 17.1 后期设备建议

| 设备         | 用途                         |

| 自准直仪       | 高精度角度校准                    |
| 激光跟踪仪      | 高精度空间点测量                   |

### 17.2 后期验证指标

| 指标       |         目标 |
| -------- | ---------: |
| yaw 误差   |     ≤ 1 角分 |
| pitch 误差 |     ≤ 1 角分 |
| roll 误差  |     ≤ 1 角分 |

### 17.3 后期核心原则

后期正式宣称 1 角分精度时，不能仅依赖标定板和视觉自洽结果，需要独立物理测量系统提供交叉验证。

## 18. 项目最终口径

本项目采用 4K 工业相机、30mm / 50mm / 100mm 三焦段镜头、300mm × 250mm 且几何精度为 0.01mm 的方向唯一高精度标定板，以及飞机受控标志点，构建基于 PnP 的飞机姿态估计实验系统。

前期阶段以标定板坐标系作为参考世界坐标系，完成相机标定、标定板 PnP、飞机受控标志点 PnP、坐标链转换、三焦段对比和姿态重复性分析。该阶段重点验证算法链路和实验系统稳定性，不直接宣称系统达到 1 角分绝对姿态精度。

后期阶段将在前期系统基础上，引入高精度转台、倾角仪、自准直仪或其他独立物理计量设备，对 PnP 姿态估计结果进行正式真值验证，最终冲击 1 角分姿态精度目标。

## 19. 参考依据

1. OpenCV 官方 PnP 文档：`solvePnP` 根据 3D–2D 对应点、相机内参和畸变参数估计物体姿态，并最小化重投影误差。
2. OpenCV 官方相机标定文档：相机标定用于求取相机内参、畸变参数、旋转和平移向量，重投影误差可用于评估标定参数精度。
3. COLMAP 官方相机模型文档：不同相机模型复杂度不同，模型选择应与畸变情况、先验标定参数和数据质量匹配。
